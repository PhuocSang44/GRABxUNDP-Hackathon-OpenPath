## Key Decisions

- **POI model added**: the original plan tracked road segments only. During session 001, an `AccessibilityPoint` model and `GET /api/points` endpoint were added to support a richer map with categorised markers (hospitals, ramps, parking, etc.). The frontend now uses points as the primary map layer.
- **MapLibre over Leaflet**: chosen for native GeoJSON clustering, vector tile support, and no licensing restrictions.
- **Marker sync on `render` event**: using `querySourceFeatures` after the `render` event (with `isMoving()` + `isSourceLoaded()` guards) instead of `moveend` eliminates the visible lag between camera movement and marker updates.
- **Icon pre-rendering**: React icons are rendered to HTML strings once at map load. Subsequent marker creation uses pure DOM — no React roots per-marker — to avoid memory pressure.
